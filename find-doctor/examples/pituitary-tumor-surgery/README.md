## Pituitary-tumor surgery example

This directory contains an HTML-dashboard export and screenshots generated on 2026-06-24 to demonstrate the `find-doctor` output:

- `01-case-overview.png`: intake summary and a care-path reminder
- `02-tiered-recommendation.png`: source-verified tiered recommendation card
- `03-action-checklist.png`: booking, records-preparation, and anti-scalper guidance
- `pituitary-tumor-surgery-example.pdf`: full example dashboard

The sample contains no patient name, contact detail, or identity-document information. It is a historical snapshot, not current medical advice or a current clinician recommendation. Listed clinician details, booking availability, prices, and care pathways can change. Confirm all information through official hospital channels and an in-person clinician; for emergencies, seek immediate local emergency care.
